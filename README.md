# 📱 BalancedBuddy

## 📌 Overview

A Kotlin-based Android app that helps users track expenses, manage budgets, and analyze spending. All data is stored locally using an SQLite (Room) database.

## 🚀 Features

* 🔐 User login and registration
* 🗂️ Create and manage expense categories
* 💸 Add expenses (amount, date, start/end time, description, category)
* 🖼️ Optional receipt photo attachment
* 🎯 Set minimum and maximum monthly budget goals
* 📋 View expenses within a selected period (with receipt access)
* 📊 View total spending per category
* 💾 Offline data storage using SQLite (RoomDB)

---

## 🛠️ Technologies Used

* Kotlin
* Android Studio
* Room Database (SQLite)
* CapCut

---

## ▶️ How to Use

1. Register or log in
2. Create categories
3. Add expenses
4. Set budget goals
5. View reports and spending
